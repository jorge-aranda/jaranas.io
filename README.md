# jaranas.io

Personal portfolio website for **Jorge Aranda**, Product Tech Lead & Backend AI Engineer.

The site is accessible at [jaranas.io](https://jaranas.io).

## Overview

This repository contains the source code for Jorge Aranda's personal landing page. It is a lightweight, single-page application built with focus on clarity, impact, and engineering craft.

## Project Structure

- `index.html`: The main entry point and only page of the website. It contains the structure, styles, and minimal client-side logic.
- `assets/`: Contains static assets like images.
- `_redirects`: Configuration file for handling redirects and domain mapping (e.g., for Netlify).

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom styles using modern CSS features (Variables, Grid, Flexbox, Gradients).
- **JavaScript**: Minimal footprint for dynamic elements (e.g., current year in footer).

## Features

- Responsive design for mobile and desktop.
- Dark theme inspired by modern engineering tools.
- Sections for About, Work Experience/Capabilities, and Contact information.
- Integration with LinkedIn and GitHub.
- Custom redirects for specific paths (e.g., `/nft`).
